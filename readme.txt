== Description ==
[english]<br>This plugin add in sidebar pages of your blog which it is pleasant to users Mail.ru. Uses jQuery. For correct work of a widget it is necessary, that the plugin "<a href="http://wordpress.org/extend/plugins/like-in-mailru">Like in Mail.ru</a>" earlier has been established. You can adjust widget display.<br>
[русский]<br>Данный плагин добавляет в сайдбар страницы вашего блога, которые нравятся пользователям Mail.ru. Используется библитотека jQuery. Для корректной работы виджета необходимо, чтобы ранее был установлен плагин "<a href="http://wordpress.org/extend/plugins/like-in-mailru">Like in Mail.ru</a>". Вы можете настроить отображение виджета.


== Installation ==
[english]<br>
1. Upload plugin files to the `/wp-content/plugins/` directory.<br>
2. Activate the plugin through the 'Plugins' menu in WordPress.<br>
3. Paste "Widget Like in Mail.ru" in your sidebar (use admin menu).<br>
4. Russian localization download <a href="http://bogutsky.ru/bogutsky/plugins/widgetlikemailru/widgetlikemailru-ru_RU.zip">this</a>.<br>
[русский]<br>
1. Загрузите файлы плагина в директорию `/wp-content/plugins/`.<br>
2. Активируйте плагин в меню администрирования.<br>
3. Поместите виджет в сайдбар (используйте меню администрирования).<br>
4. Русскую локализацию можно скачать <a href="http://bogutsky.ru/bogutsky/plugins/widgetlikemailru/widgetlikemailru-ru_RU.zip">здесь</a>.
== Frequently Asked Questions ==
[english]
= Why a widget empty? =
Because the plugin "<a href="http://wordpress.org/extend/plugins/like-in-mailru">Like in Mail.ru</a>" isn't established.<br>
[русский]
= Почему виджет пустой? =
Потому что у вас не установден плагин "<a href="http://wordpress.org/extend/plugins/like-in-mailru">Like in Mail.ru</a>".
